NOME
----

`ss` - ss para Investigar Sockets

SYNOPSIS
--------

`ss` [`-bar`] [`-c` *config-file* ] *file* ...

DESCRIÇÃO
---------

`ss` é um programa usado para inspecionar melhor as informações de conexões de rede.

OPÇÕES
------

* `-4`:
  Usado para exibir os sockets IPv4.

EXEMPLOS
--------

Coloque os exemplos aqui:

   `$> ss -4 '`
