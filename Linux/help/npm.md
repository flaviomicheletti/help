OPÇÕES
------

* `npm config set prefix "/lib/node-modeules"`:
	Define o local onde será armazenada 

* 'npm -g request
	Instala o modulo request globalmente.

* `npm audit`:
	Faz um check por vulnerabilidades.

* `npm audit fix`:
	Resolve as vulnerabilidades encontradas.
