lsvg(1) -- lsvg para minhas man pages v1.0
===============================================

NOME
----

`lsvg` - lsvg exibe as informações do VG.

SYNOPSIS
--------

`lsvg` [`-bar`] [`-c` *config-file* ] *file* ...

DESCRIÇÃO
---------

`lsvg` é usado para exibir informações sobre o VG. No AIX o rootvg é o VG padrão do sistema.
